# ☕ Java Core Journey - Lộ Trình Tự Học & Nâng Cao

Chào mừng bạn đến với kho lưu trữ **Java Core**! Đây là dự án cá nhân được tôi xây dựng để ghi chép lại toàn bộ quá trình **tự học**, thực hành và đúc kết kinh nghiệm lập trình Java từ con số không. 

Dự án này không chỉ là nơi lưu trữ source code mà còn là cuốn sổ tay kỹ thuật số, giúp tôi củng cố tư duy lập trình và làm nền tảng vững chắc để tiến tới các kiến thức nâng cao hơn như Design Patterns hay các framework mạnh mẽ.

## 🎯 Mục Tiêu Đạt Được

- **Nắm chắc nền tảng Java:** Biến, kiểu dữ liệu, vòng lặp, mảng, xử lý chuỗi, Collection Framework.
- **Tư duy Hướng đối tượng (OOP):** Hiểu sâu và vận dụng tốt 4 tính chất (Đóng gói, Kế thừa, Đa hình, Trừu tượng).
- **Nguyên lý thiết kế:** Tiếp cận và thực hành các nguyên lý mã sạch (Clean Code) và các nguyên lý **SOLID**.
- **Công cụ & Kỹ năng:** Thành thạo sử dụng IntelliJ IDEA, kỹ năng gỡ lỗi (debug) và quản lý mã nguồn bằng Git.
